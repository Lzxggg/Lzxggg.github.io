微小细节

fortran :

  • 每一条语句后不需要加分号
  • fortran的注释：! C语言的注释: //
  • 如果续行可以添加&符号
  • fortran的双冒号 :: 作用 声明一个变量后可以直接在后边赋值 不然像integer a = 10 是不对的语法 应该是integer :: a = 10
  • fortran是不区分 大小写字母的 然而 C 是区分的 所以混合编程的时候 函数名尽量别用驼峰命名法 即 ：maxNum 这种 尽量纯小写
  • 当第一条执行命令开始的时候 声明便不可用了后边再有声明会报错 所以执行命令开始前要定义 声明好所有的变量

C：每一条语句后都要加分号 不然编译不通过

数据类型

fortran :

integer 相当于C语言的 int

real :

  • real(4) 相当于C语言的float
  • real(8) 相当于C语言的double

character ：

相当于C语言的 char 但是又和char 有所不同

character (num)

当num == 1 或者没有括号 charater :: b时 ，说明 b 是一个字符

当num >= 2 时 可以作为一个字符串

字符串的连接符：//

logical:

相当于 C 的Boolean

.true. 相当于 true

.false. 相当于false

complex：
