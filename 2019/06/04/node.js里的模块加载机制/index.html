node.js里的模块加载机制

1.看缓存里是否有 有的话优先载入

2.优先查看是否是核心模块（例如：fs http 等模块） 如果是直接载入

3.如果是第三方模块

  • 他会到当前目录的node_modules下面寻找你所引入的 模块名称
  • 例如你引入的是一个 requrie('body-parser') 他就去node_modules目录下面找body-parse.

  • 如果查找得到那么就会找到所查找目录下寻找 package.json 并在其中寻找main入口，如果没有main入口将会寻找 index.js

  • 上述过程一直到找到index.js才算引入成功 或者 找到main入口并且文件存在

  • 如果上述过程都找不到则去文件的上一层目录寻找 一直找到根目录还找不到的话 报错！
