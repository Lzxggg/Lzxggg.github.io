MongoDB 数据库 在 node.js 中的应用

MongoDB 数据库：

  • 和 mysql 相比 MongoDB 更加简单一些 ，不用去定义一个表中有什么字段，直接插入即可，所以每一个元组的数据可长可短，不统一，看似并不能去适应具有一定关系的开发，但是在 mongoose 中的 API 可以去定义这些东西，而 mysql 在插入元组之前必须定义好所有的字段以及约束关系
