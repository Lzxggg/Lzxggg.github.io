node 中的 path 核心模块的 API 部分 总结

node 中的 path 核心模块的 API 部分 总结

  • path.basename
    • 获取一个路径的文件名
  • path.dirname
    • 获取一个路径的目录部分
  • path.extname
    • 获取一个路径中的扩展名部分
  • path.parse
    • 把一个路径传换成对象
      • root 根路径
      • dir 目录
      • base 包含后缀名的文件名
      • ext 后缀名
      • name 不包含后缀名的文件名
  • path.join
    • 当使用路由拼接时 使用这个方法
  • path.isAbsolute 判断一个路径是否是相对路径

node中的其他成员 (直接使用)

举例 使用 c : \ a \ b \ c.js

  • __dirname : 获取这个文件的绝对路径 （不包含这个文件的名字)
  • __filename ： 获取这个文件的绝对路径（包含这个文件的名字）

应用： 在读文件的操作的时候 尽量不要使用相对路径 因为一旦换个目录执行此js文件 便会报错 因为它会到当前的目录下面寻找 所以我们不能写死 要用__dirname
