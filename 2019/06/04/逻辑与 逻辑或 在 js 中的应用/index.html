逻辑与 逻辑或 在 js 中的应用

逻辑或 || ： var a = 表达式1 || 表达式2

　　　　　　表达式1　　　　表达式2　　　　a取值

　　　　　　 1 　　　　　　0　　　　　　表达式1结果值

　　　　　　 1　　　　　　　 1　　　　　　表达式1结果值

　　　　　　 0　　　　　　　 1　　　　　　表达式2结果值

　　　　　　 0　　　　　　　 0　　　　　　表达式2结果值

逻辑与 && ： var a = 表达式1 && 表达式2

　　　　　　表达式1　　　　表达式2　　　　a取值

　　　　　　 1 　　　　　　0　　　　　　表达式2结果值

　　　　　　 1　　　　　　　 1　　　　　　表达式2结果值

　　　　　　 0　　　　　　　 1　　　　　　表达式1结果值

　　　　　　 0　　　　　　　 0　　　　　　表达式1结果值
