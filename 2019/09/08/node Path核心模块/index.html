node Path核心模块

Path路径操作模块

path是一个核心模块，专门用来操作路径的。

  • path.basename(‘c:/a/b/c/index.js’) //index.js

    获取路径的文件名包含后缀名

    path.basename(‘c:/a/b/c/index.js’,’.js’) //index

    获取路径文件文件名

  • path.dirname(‘c:/a/b/c/index.js’) // ‘c:/a/b/c’

  • path.extname(‘c:/a/b/c/index.js’) // 获取一个文件的扩展名 .js

  • path.isAbsolute(‘c:/a/b/c/index.js’) //true

    判断是否是绝对路径

  • path.parse()

    是前面几个的组合

    • root 根路径
    • dir 目录
    • base 包含后缀名的文件名
    • ext 后缀名
    • name 不包含后缀名的文件名

  • path.join

    当你需要进行路径拼接的时候 推荐使用这个方法

Node 中的非模块成员

在每个模块中 除了require,exports等模块相关API之外，还有两个特殊的成员

  • __dirname 可以用来获取当前文件模块所属目录的绝对路径
  • __filename 可以用来获取当前文件的绝对路径
