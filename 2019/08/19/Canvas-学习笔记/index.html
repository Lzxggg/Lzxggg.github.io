在html里面的结构

canvas 就是使用 js 来进行绘图 所以如果使用 Canvas 一定要先获取这个 Canvas

常用API和属性总结

Api:

Api名称 作用及应用
getContext() 初始化画布 获取画布的一系列工具， ctx.getContext(‘2d’)
beginPath() 起始一条路径，或重置当前路径
closePath() 用于闭合封闭图形
moveTo() 把路径移动到画布中的指定点，不创建线条
lineTo() 添加一个新点，然后在画布中创建从该点到最后指定点的线条
stroke() 绘制已定义的路径
fill() 填充当前绘图（路径)
rect() 创建一个矩形,参数分别是 x,y的坐标 和 矩形的大小 (20,20,100,200)

属性:

属性 描述
fillStyle 填充颜色 值：正常16进制颜色
lineWidth 改变线条的宽度
strokeStyle 画笔的颜色
lineCap 用于改变直线两端的样式 值： butt:默认 无样式;square:两端跟方块;round:两端圆角
lineJoin 用于改变拐点的样式 值：bevel:斜角;round:圆角;miter:默认创建尖角

关于在 canvas 中像素点的问题

在 canvas 中 如果你绘制一个lineWidth 为 1 的线条 他其实是

两个1px的像素 合在一起 它在每个像素格子里占一半 所以画出来显得很虚

我们可以在它的Y轴加或者减掉0.5这样就会好起来
