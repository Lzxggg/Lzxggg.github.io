let 和 const

  1. ES6新更新了块级作用域的存在，let和const在块级作用域内声明的变量在作用域外都无法访问

  1. let和const不能重复定义（在同一个作用域内 如果在不同的作用域内可以定义重复的变量）

  2. const所定义的是常量 他所定义的变量值不允许修改 所以声明变量必须同时进行初始化

  3. let和const 不存在预解析(什么是预解析 就是把函数 变量的声明提到最前面进行 赋值是执行到相应的行数的时候进行赋值 如果之前没赋值就打印 那么就是 undefined)

字符串新增方法

  1. includes() //用来判断一个字符串是否含有另一个字串 如果有返回true 否则 false

  2. endsWith() //用来判断一个字符串是否由一个指定的字符串结尾

  3. startsWith() //用来判断一个字符串是否由一个指定的字符串开头

  4. 模板字符串 用``来书写模板字符串 非常像art-template
